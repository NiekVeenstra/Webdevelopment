Opdracht 1.

1. X::before{} hierin kan X een element zijn en daarbij word de before toegepast voor de desbeetreffende element X. 2. X::after() hierin is X een een element waarna de content van de after toegepast word. 3. X::first-letter{} hierin word er style toegepast op de eerste letter op de tekst die voorkomt in het X element. 4. ::selection{} past een style toe op de content die door de gebruiker geselecteerd word met zijn muis.
